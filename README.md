# Weather App 🌤

Интерактивное приложение для отслеживания погоды в реальном времени. Проект демонстрирует навыки работы с внешними API, обработки JSON-данных и динамического обновления интерфейса.

## 🛠 Технологический стек

- **HTML5:** Семантическая разметка интерфейса.
- **CSS3:** Flexbox-верстка, стилизация погодных виджетов и адаптивный дизайн.
- **JavaScript (ES6+):** - Работа с **Fetch API** для получения данных.
    - Асинхронные функции (`async/await`).
    - Динамическое манипулирование DOM.
- **API:** [OpenWeatherMap API](https://openweathermap.org/api) (или аналог) для получения актуальных данных.

---

## 🚀 Как запустить проект

1. Откройте папку проекта в **VS Code**.
2. Запустите расширение **Live Server**.
3. Введите название города в поле поиска, чтобы получить прогноз.

> **Важно:** Для работы приложения требуется активный API-ключ. Убедитесь, что он корректно прописан в вашем JS-файле.

---

## 📱 Особенности реализации

- **Real-time Data:** Получение актуальной температуры, влажности и описания погоды в реальном времени.
- **Dynamic UI:** Автоматическая смена иконок или фонов в зависимости от погодных условий.
- **Error Handling:** Обработка ошибок при вводе несуществующего города или проблемах с сетью.
- **Responsive:** Приложение корректно отображается как на десктопах, так и на мобильных устройствах.
